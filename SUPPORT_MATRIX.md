# Support Matrix

版本：`0.1.0-alpha.1`

## 1. 正式支持

- Public web research
- Known-URL extraction
- Safe-first default routing
- Supported static and site-specific extractors
- `Codex`
- `Claude Code`
- macOS alpha host environment

## 2. 高级模式支持

以下能力需要用户显式开启高级模式：

- Isolated browser-profile reads for limited public dynamic pages
- Guided advanced dependency preparation
- Browser-assisted public-page rendering when static routes are insufficient

## 3. 实验性支持

- `Bilibili` public video pages through the experimental `Tier 1.5` backend
- Standalone repository export and publish-prep flow

## 4. 明确不承诺

- Logged-in site reading
- OAuth flows
- Stable access to `x.com`
- Stable access to `platform.openai.com`
- Use of the user's daily browser profile
- Windows / Linux first-version support

## 5. 站点层正式支持集合

- Reddit public feeds and threads
- Stack Overflow and selected Stack Exchange sites
- arXiv abstract pages
- Hugging Face public model pages
- selected official docs sites
- YouTube watch pages
- BBC article pages
- WeChat article pages
- Wikipedia article pages

## 6. 结果解释

- `Supported`: part of the formal product promise for this alpha line
- `Advanced-only`: supported only after explicit mode switch
- `Experimental`: usable in some cases, but not yet a release promise
- `Deferred / Unsupported`: outside the current product line
